REQUISITOS
- NetBeans IDE 30 o superior
- JDK 21 o superior

INSTRUCCIONES DE EJECUCION

1. Descomprima el archivo Grupo_XX.zip

2. Abra NetBeans
   File -> Open Project -> seleccione la carpeta "GestionReclamos"

3. Compile el proyecto
   Clic derecho en el proyecto -> Clean and Build

4. Ejecute el programa
   Presione F6 o el boton verde Play

5. El menu interactivo aparecera en la consola inferior

FUNCIONALIDADES DEL SISTEMA
- Registrar, modificar, eliminar y consultar reclamos
- Busqueda binaria por codigo (O log n)
- Busqueda secuencial por RUT (O n)
- Ordenamiento por codigo, prioridad, fecha limite, fecha ingreso y tipo
- Cola de atencion de reclamos pendientes
- Historial de cambios por reclamo
- Arbol BST y Arbol AVL con recorridos InOrden, PreOrden y PostOrden
- Filtro automatico de vencimientos proximos
